## 🎯 核心功能

### 1. 飞书命令支持

| 命令 | 说明 | 示例 |
|------|------|------|
| `/hr send_interview` | 发送面试通知 | `/hr send_interview 陈琳 初试 下周一 10 点` |
| `/hr schedule` | 安排面试 | `/hr schedule 张伟 复试 周三 2 点` |
| `/hr urgent` | 加急面试 | `/hr urgent 李悦 明天` |
| `/hr status` | 查看状态 | `/hr status 陈琳` |
| `/hr confirm` | 确认面试 | `/hr confirm 陈琳 参加` |
| `/hr reschedule` | 改期 | `/hr reschedule 陈琳 下周二 10 点` |

### 2. 自然语言支持

```
发送面试通知给陈琳
安排陈琳面试，下周一上午 10 点
加急！给张伟发面试通知
陈琳确认参加面试
李悦需要改期，下周二可以吗
```

### 3. 飞书卡片交互

**草稿确认卡片：**
```
┌─────────────────────────────────┐
│ 📧 面试通知邮件草稿             │
├─────────────────────────────────┤
│ 候选人：陈琳                    │
│ 岗位：新媒体内容运营            │
│ 时间：3 月 16 日 10:00           │
│                                 │
│ [✅ 发送] [✏️ 修改] [❌ 取消] │
└─────────────────────────────────┘
```

**发送成功卡片：**
```
┌─────────────────────────────────┐
│ ✅ 面试通知已发送               │
├─────────────────────────────────┤
│ 候选人：陈琳                    │
│ 状态：⏳ 等待确认               │
│ 截止：3 月 14 日                 │
│                                 │
│ [查看记录]                      │
└─────────────────────────────────┘
```

### 4. 多维表格联动

**自动更新字段：**
- 当前状态 → "面试中"
- 面试通知发送时间
- 面试确认状态
- 面试轮次

**自动创建记录：**
- 面试记录表新建记录
- 关联候选人
- 记录面试官、时间、形式

### 5. 邮件模板

| 模板 | 适用场景 |
|------|----------|
| 模板 1 | 初试通知（HR 面） |
| 模板 2 | 复试通知（业务面） |
| 模板 3 | 终试/Offer 面谈 |
| 模板 4 | 紧急/加急面试 |

### 6. 自动提醒

| 时间点 | 动作 |
|--------|------|
| 发送后 24h 未确认 | 第一次提醒 |
| 发送后 48h 未确认 | 第二次提醒 |
| 截止前 2h | 最后提醒 |
| 面试前 1 天 | 面试提醒 |
| 面试前 2h | 再次提醒 |
