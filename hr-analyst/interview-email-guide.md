# 面试通知邮件发送功能 - 使用指南

## 📋 功能概述

HR Analyst 可以自动向推荐度最高的候选人发送面试通知邮件，支持：
- 自动选择最佳候选人
- 智能匹配邮件模板
- 填充公司和面试信息
- 生成草稿待确认
- 确认后自动发送

---

## 🚀 快速开始

### 基本命令

```
# 给排名第一的候选人发面试通知
给排名第一的候选人发面试通知
岗位：[岗位名称]
时间：[面试时间]
形式：[现场/视频]

# 给指定候选人发面试通知
发送面试通知给 [候选人姓名]
时间：[面试时间]

# 安排面试
安排 [候选人姓名] 面试
轮次：[初试/复试/终试]
时间：[时间]
```

## ⚙️ 配置说明

### 1. 配置公司信息

编辑 `company-config.yaml`：

```yaml
公司:
  名称：杭州创想科技有限公司
  地址：杭州市西湖区 XX 路 XX 号 XX 大厦 18 楼
  官网：https://www.example.com
  
HR 团队:
  招聘负责人:
    姓名：王晓雅
    邮箱：wangxiaoya@example.com
    电话：0571-88888881
```

### 2. 配置面试地点

```yaml
面试配置:
  现场:
    地址：杭州市西湖区 XX 路 XX 号 XX 大厦
    楼层：18 楼会议室
    交通：地铁 1 号线 XX 站 B 出口
    
  视频:
    平台：飞书会议
    会议室：预留会议号
```

### 3. 配置邮件发送

```yaml
邮件配置:
  SMTP:
    服务器：smtp.example.com
    端口：587
    发件人：hr@example.com
    密码：[SMTP 密码]
```

## 🔐 安全与确认

### 发送前确认

所有邮件发送前都会：
1. 生成草稿预览
2. 等待用户确认
3. 确认后才发送

### 确认命令

```
发送      → 立即发送邮件
修改      → 编辑邮件内容
取消      → 放弃发送
重发      → 重新发送给同一人
```

### 发送记录

每封邮件都会：
- 记录到飞书多维表格
- 保存邮件副本到 `interviews/` 目录
- 更新候选人状态为"面试中"

## ❓ 常见问题

### Q: 可以批量发送面试通知吗？

A: 可以。使用命令：
```
给前三名候选人发面试通知
时间范围：下周一到周三
形式：现场面试
```

### Q: 候选人要求改期怎么办？

A: 回复改期请求后，HR Analyst 会：
1. 解析候选人期望时间
2. 检查面试官日历
3. 生成新的面试通知草稿
4. 确认后发送

### Q: 可以自定义邮件模板吗？

A: 可以。编辑 `templates/interview-email-templates.md` 添加或修改模板。

### Q: 邮件发送失败怎么办？

A: HR Analyst 会：
1. 检测发送状态
2. 失败时通知 HR
3. 提供重试选项

## 📊 数据统计
