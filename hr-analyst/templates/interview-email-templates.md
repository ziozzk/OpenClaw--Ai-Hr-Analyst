# 面试通知邮件模板

## 模板 1：初试通知（标准版）

```
主题：【面试邀请】{公司名称} - {岗位名称} - {候选人姓名}

尊敬的 {候选人姓名}：

您好！

感谢您关注 {公司名称} 的 {岗位名称} 职位。经过对您简历的评估，
我们认为您的背景与该岗位高度匹配，现诚挚邀请您参加初试。

┌─────────────────────────────────────────────────────────┐
│  📋 面试信息                                            │
├─────────────────────────────────────────────────────────┤
│  岗位：{岗位名称}                                       │
│  面试轮次：初试                                         │
│  面试形式：{现场面试 / 视频面试}                        │
│  时间：{日期} {时间}                                    │
│  时长：约 60 分钟                                        │
├─────────────────────────────────────────────────────────┤
│  📍 面试地点（现场面试）                                 │
│  地址：{详细地址}                                       │
│  楼层：{楼层/会议室}                                    │
│  交通：{地铁/公交路线}                                  │
├─────────────────────────────────────────────────────────┤
│  💻 面试链接（视频面试）                                 │
│  平台：{腾讯会议/飞书会议/Zoom}                         │
│  会议号：{会议号}                                       │
│  密码：{密码}                                           │
├─────────────────────────────────────────────────────────┤
│  👥 面试官                                              │
│  {面试官姓名} - {职位}                                  │
├─────────────────────────────────────────────────────────┤
│  📝 面试准备                                            │
│  • 请准备 3-5 分钟的自我介绍                             │
│  • 携带纸质简历一份（现场面试）                         │
│  • 准备 1-2 个代表作品或项目案例                         │
│  • 提前 5 分钟到达/进入会议室                           │
└─────────────────────────────────────────────────────────┘

【确认回复】
请您在 {截止日期} 前回复本邮件确认是否参加。
如时间不便，请告知您方便的时间段，我们将尽力协调。

【联系方式】
联系人：{HR 姓名}
电话：{联系电话}
邮箱：{联系邮箱}

期待与您的交流！

此致
敬礼

{公司名称} 人力资源部
{日期}

---
{公司名称}
公司地址：{公司注册地址}
公司官网：{公司网站}
```

## 模板 4：紧急/加急面试通知

```
主题：【紧急面试邀请】{公司名称} - {岗位名称} - {候选人姓名}

尊敬的 {候选人姓名}：

您好！

{公司名称} 对您的简历非常感兴趣，希望能尽快安排面试交流。

┌─────────────────────────────────────────────────────────┐
│  📋 面试信息                                            │
├─────────────────────────────────────────────────────────┤
│  岗位：{岗位名称}                                       │
│  时间：{日期} {时间}（{星期 X}）                        │
│  形式：{现场/视频}                                      │
└─────────────────────────────────────────────────────────┘

【特别说明】
本次面试为加急安排，如您时间不便，请告知我们您最早可参加面试的时间。

【快速确认】
回复本邮件或短信确认：
• 确认参加：回复"确认 + 姓名"
• 需要改期：回复"改期 + 可选时间"

【联系方式】
HR {姓名}：{手机号}

感谢配合！

{公司名称} 人力资源部
{日期}
```

## 变量说明

| 变量 | 说明 | 示例 |
|------|------|------|
| `{公司名称}` | 招聘公司全称 | 杭州 XX 科技有限公司 |
| `{岗位名称}` | 招聘职位 | 新媒体运营专员 |
| `{候选人姓名}` | 候选人姓名 | 李悦 |
| `{日期}` | 面试日期 | 2026 年 3 月 15 日 |
| `{时间}` | 面试时间 | 下午 14:00 |
| `{面试形式}` | 现场/视频 | 现场面试 |
| `{详细地址}` | 面试地址 | 杭州市 XX 区 XX 路 XX 号 |
| `{面试官姓名}` | 面试官名字 | 张经理 |
| `{HR 姓名}` | HR 联系人 | 王小姐 |
| `{联系电话}` | 联系电话 | 0571-88888888 |

---

## 使用方式

**在飞书/聊天中触发：**
```
发送面试通知给李悦
岗位：新媒体运营专员
时间：明天下午 2 点
形式：视频面试
```

**HR Analyst 会自动：**
1. 选择合适模板
2. 填充候选人信息
3. 生成邮件草稿
4. 等待确认后发送
